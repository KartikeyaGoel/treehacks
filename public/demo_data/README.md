# Demo sleep data

| File | Label | Description |
|------|--------|-------------|
| **demo_healthy_30days.csv** | Healthy | 30 days of stable, good sleep. Recent 7 days match baseline → analysis shows normal variation (low z-scores, stable SHDI). Use to demo “no significant deviation.” |
| **demo_sick_30days.csv** | Sick | Same person, but last ~8 days show clear deterioration (lower efficiency, deep, REM; higher awakenings). Mimics illness/stress onset. Analysis shows significant deviations (~±4σ) and elevated SHDI. |
| demo_ultra_dramatic.csv | Dramatic | Very large contrast (perfect baseline then severe crash). Best for high-impact demos. |
| demo_dramatic.csv | Moderate | Moderate deterioration in final week. |
| fitbit_sample.csv / fitbit_30days.csv | Fitbit format | Sample Fitbit-export data. |

**Quick pick:** Use **demo_healthy_30days.csv** for a “healthy” result; use **demo_sick_30days.csv** for a “sick” / deterioration result.
